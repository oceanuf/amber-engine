## 🔍 G12-G15核心数据需求

### **G12 能量潮汐评委**
**核心需求**: 板块级资金流共振检测
**具体数据**:
- 全市场板块分类 (行业/概念板块)
- 板块内个股资金流向 (主力/大单/中单/小单资金)
- 板块资金流历史序列 (日/周/月)
- 资金流同步性统计指标

### **G13 政策语义向量家**
**核心需求**: LLM政策文档语义分析
**具体数据**:
- 国务院/部委政策公文 (标题+内容)
- 地方政府产业规划文件
- 证监会/交易所公告
- 央行货币政策报告
- 政策发布时间与权威级别

### **G14 波动率挤压2.0**
**核心需求**: 期权隐含波动率(IV)差值方向预判
**具体数据**:
- 上证50ETF期权隐含波动率
- 沪深300股指期权隐含波动率
- 看涨/看跌期权IV差值
- 期权成交量与持仓量
- 历史波动率(HV)数据

### **G15 跨市场关联评委**
**核心需求**: 全球流动性对A股前瞻性影响
**具体数据**:
- 离岸人民币汇率(USDCNH)实时数据
- 富时A50期货价格
- 美国10年期国债收益率
- 美元指数(DXY)
- VIX恐慌指数
- CRB商品指数

## 📋 Tushare Pro数据能力评估

### ✅ **已确认支持的数据**

| 数据类型 | Tushare Pro接口 | 数据质量 | 备注 |
|----------|-----------------|----------|------|
| **个股资金流向** | `moneyflow` | 🟢 高 | 支持主力/大单/中单/小单资金分类 |
| **板块资金流向** | `moneyflow_hsgt` | 🟡 中 | 仅限北向资金，非全市场板块 |
| **行业分类** | `index_classify` | 🟢 高 | 支持申万/中信行业分类 |
| **概念板块** | `concept` | 🟡 中 | 概念板块分类，更新频率一般 |
| **宏观经济** | 多个接口 | 🟢 高 | CPI/PPI/M2/利率等 |
| **汇率数据** | `fx_daily` | 🟢 高 | 主要货币对汇率 |

### ❌ **不支持的/有限支持的数据**

| 数据类型 | Tushare Pro支持情况 | 替代方案建议 |
|----------|---------------------|--------------|
| **政策公文** | ❌ 不支持 | 需要外部API (政府网站爬虫/第三方政策数据库) |
| **期权IV数据** | ❌ 不支持 | 专业金融数据商 (Wind/Choice/聚宽) |
| **A50期货** | ❌ 不支持 | 国际金融数据API (Bloomberg/Reuters) |
| **VIX恐慌指数** | ❌ 不支持 | 国际金融数据API |
| **板块资金流(全市场)** | ⚠️ 有限 | 需自行聚合个股资金流向 |

### 🔄 **可替代方案**

#### 1. **政策文档数据源**
- **中央人民政府网站**: http://www.gov.cn (爬虫)
- **各部委官方网站**: 发改委/证监会/央行等
- **第三方政策数据库**: 同花顺iFinD/Wind政策库
- **LLM实时搜索**: Perplexity API + 网页抓取

#### 2. **期权数据源**
- **上证所/深交所官方数据**: 期权交易数据API
- **专业金融平台**: 聚宽JoinQuant/米筐RiceQuant
- **付费数据商**: Wind/Choice金融终端

#### 3. **全球市场数据**
- **外汇**: 雅虎财经/Yahoo Finance API
- **期货**: Investing.com/新浪财经
- **国际指数**: 彭博社Bloomberg/路透社Reuters

## 🚨 风险与挑战

### **技术风险**
1. **数据源稳定性**: 政府网站反爬虫策略，API接口变更
2. **数据延迟**: 国际数据源延迟可能影响实时性
3. **数据质量**: 不同数据源格式/质量不一致

### **成本风险**
1. **付费数据源**: Wind/Choice等专业数据年费较高
2. **API调用限制**: 免费API有调用频率限制
3. **存储成本**: 政策文档/期权数据存储需求大

### **合规风险**
1. **数据使用授权**: 确保数据源使用合规
2. **爬虫合法性**: 政府网站爬虫需遵守robots.txt
3. **数据安全**: 敏感金融数据存储与传输安全
