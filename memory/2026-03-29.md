## 📋 技术标准核心要点

### 设计准则 (The Core Principles)
1. **积木式模组化** - Lego-style独立模块
2. **下层不依赖上层** - Bottom-up Hierarchy
3. **故障隔离** - 单一模组故障不引发系统崩溃

### 业务链条 (8个标准化节点)
1. Ingest → 2. Cleaner → 3. Storer → 4. Analyzer → 5. Synthesizer → 6. Targeter → 7. Advisor → 8. Notary

2. **表现层** (CSS/HTML) - 与逻辑层解耦
3. **数据层** (多级) - 静态MD + 动态JSON
