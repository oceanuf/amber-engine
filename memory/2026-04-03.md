## 🔍 主编提出的战略补充模块

### 1. **深蓝探针** (Probe Module)
**问题**: 当前Ingest模块解决了"提取哪些数据"，但缺少"提取谁的数据"  
**功能**: 用已知找相似的未知 (如：从000681分析行业/板块/概念的潜力标的)  
**位置**: Ingest之前，数据预筛选层  
**输入**: 已知标的 + 分析维度 (行业/板块/概念)  
**输出**: 候选标的列表 → 传递给Ingest模块

### 2. **全球资讯哨兵** (Sentinel Module)  
**问题**: 探针只能被动出击，需要主动引导  
**功能**: 捕捉财经资讯，主动发现投资机会  
**位置**: 探针之前，信息感知层  
**输入**: 财经新闻、公告、研报等资讯流  
**输出**: 线索列表 → 传递给探针模块

### 3. **演武场** (Arena Module)
**问题**: Targeter只输出命中标的，缺乏实际交易验证  
**功能**: 量化自动交易，至少30天锁仓，检验合成算法  
**位置**: Targeter之后，Advisor之前  
**输入**: Targeter的命中标的 + Synthesizer的交易信号  
**输出**: 模拟/实际交易记录 + 绩效评估 → 反馈给算法优化
