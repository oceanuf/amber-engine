## 演武场的工作原理

我们开发了G1-G11种算法，这11位算法就像是11位评委，他们最终决定了标的是否有进入演武场的机会。

哨兵与探针的角色，就是挖掘标的，相当于推荐候选人。

G1-G11的角色就是评委，相当于对推荐的标的，从自己的角度进行评分。

每种算法都有独立的评分规则，针对某一标的（候选人）进行独立评分，最高可以打满分，最低可以打0分。

标的（候选人）经过了11位评委打分后，获得了初始的分值，这个分值最终决定了标的是否有进入演武场的机会。

如果标的进入了演武场，那么根据我们的反馈机制，哨兵会赢得一些入门经验分。

G1-G11在给候选标的打分的时候，由于它们各自的动态权重不同，所以标的最终分值计算公式应该是：G1得分XG1算法权重+G2得分XG2算法权重+...

G1-G11的权重机制，是我们预先设定并根据标的在演武场的表现动态调整的。

G1-G11，决定了标的是否有入局的机会, 我们会给标的设定基本的入门分值，达到特定分值即可进入演武场并获得同等的初始资金额度。

所有的成功入选演武场的标的，初始资金额度相同，都是10万元。

标的入选演武场的当天的收盘价，即为该标的买入价；

标的进入演武场后每天都会有动态分值，动态分值与标的收益表现直接挂钩。

如果某一算法在标的入门的时候给予某一标的极高的推荐分，现阶段该只标的收益表现又相当亮眼，这也会正向给某一算法增加动态权重分值。反之，亦然。

如果标的在演武场表现欠佳，触及清仓标准，那么标的直接退出，同时会影响G1-G11相应的权重分值。

标的进入演武场后，会遵行演武场的交易策略，收益触及相应的策略，会进行对应的减仓止盈或止损。不管止盈或者止损都会动态影响G1-G11的权重分值。

## 📋 一、模型概述与核心逻辑解析

### 1.1 模型架构理解

您提出的演武场模型是一个**多层筛选 + 动态反馈 + 优胜劣汰**的完整系统：

```
感知层(哨兵) → 搜索层(探针) → 生产层(1-5模块) → 实战层(演武场) → 输出层(6-8模块)
```

**核心创新点**:
1. **评委制评分**: G1-G11作为独立评委，避免单一算法偏差
2. **动态权重调整**: 算法权重基于实战表现动态优化
3. **双层竞争机制**: 正式队(30只) + 替补队(预备)的竞争结构
4. **收益对比基准**: 以510300(沪深300)为基准，强调超额收益
